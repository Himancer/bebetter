🔐 SECURITY FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Password Hashing
   • bcrypt with salt rounds
   • Passwords never stored in plain text
   • One-way hashing (no decryption)

✅ JWT Authentication
   • Tokens signed with secret key
   • Token expiration support
   • Stateless authentication (no sessions)

✅ Input Validation
   • Pydantic models enforce types & constraints
   • SQL injection prevention via ORM
   • Email validation

✅ CORS Protection
   • Cross-origin requests controlled
   • Only frontend domain allowed (configurable)

✅ XSS Protection
   • React auto-escapes HTML
   • No innerHTML usage

✅ Rate Limiting
   • AI Chat: 10 messages/day per user
   • Prevents abuse
